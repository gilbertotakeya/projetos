acesse a pasta AnimalFinder pelo prompt de comando
informe o comando rails server
o projeto abrirá com o cadastro de animais
caso deseje cadastrar o dono, informe apos o localhost:<porta>/donos 
e abrirá a opção para cadastro de donos

Obs: Infelizmente tive algumas dificuldades por ser o primeiro projeto em
Ruby, precisaria de mais uma semana para deixar o projeto com uma interface
amigavel e rodando da forma que deve ser.
